The Research Skill

Replaces manual Gemini sessions. Produces strategic briefings with actionable hooks.

Input

lead_name: "Alex Hormozi"
company: "Acquisition.com"
youtube_url: "@AlexHormozi"
context: "AI retention for course creators"

Process

  • ✓ Tavily search: background, news, interviews
  • ✓ YouTube scrape: recent videos + transcripts (Apify)
  • ✓ Content analysis: values, philosophy, communication style
  • ✓ Hook generation: 3+ actionable openers
API: Tavily

search_depth: "advanced" | include_answer: true

Output: Strategic Briefing

# Alex Hormozi — Account Briefing

## Philosophy
"Give away the secrets, sell the implementation"
Anti-guru, proof-focused, aggressive transparency

## Top Hooks
1. Reference "$100M Offers" framework

3. Connect to "volume negates luck"

## Don't Say
- Fluffy language, vague promises
- "Just wanted to reach out"
- Generic AI buzzwords

working hard enough"

most programs we see: 15% finish rate..."
Deep Dive

The Cold Email Skill

Inspired by Lemlist & Instantly. Short, punchy, personalized.

Core Rules

  • 75 words max — Every word earns its place
  • No generic openers — Skip "Hey, reaching out..."
  • Specific > vague — "267k members = ~$80K" beats "$50K-$300K"
  • One clear CTA — "Want me to show you?"
  • Pattern interrupt — First line stops the scroll

5-Email Sequence

Email Day Purpose
10Hook + Problem
23Value + Proof
37Social Proof
410Risk/FOMO
514Hail Mary

Example Output

To: alex@acquisition.com
Subject: Alex — quick question about Gym Launch
Alex — watched your breakdown on retention curves.

Most programs we see: 15% finish rate. The rest ghost after week 2.

With 267k members, that's roughly $80K sitting on the table.

We built something that changes that. No fluff — just AI that keeps people engaged 24/7.

Worth a 10-min call to see if it fits?
Word count: 67 ✓
Integration

Lemlist API

Campaign creation, lead upload, sequence automation.

Key Endpoints

# Base: https://api.lemlist.com/api
# Auth: Basic (empty user, API key as pass)

GET  /campaigns          # List all
POST /campaigns          # Create new
POST /campaigns/{id}/leads   # Add lead
POST /campaigns/{id}/start   # Launch
POST /enrich             # Find email
POST /enrich/bulk        # Batch (500 max)

Variables Available

# Standard
{{firstName}} {{lastName}} {{email}}
{{companyName}} {{phone}} {{linkedinUrl}}

# Custom (from lead data)
{{communityName}} {{memberCount}}
{{calculatedLTV}} {{completionRate}}

Campaign Creation Flow

1. Create campaign
POST /campaigns {"name": "Skool AI Q2"}
2. Add email sequence
POST /campaigns/{id}/sequences
3. Upload leads
POST /campaigns/{id}/leads (batch)
4. Start campaign
POST /campaigns/{id}/start ✅
⚠️ Rate Limit: 10 requests/second per team
